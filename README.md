# Centown

## Required Minecraft Installation

* Minecraft 1.16.5
* [Fabric Loader](https://fabricmc.net/use/installer/)

## Required Mods

Please use the 1.16.5 versions of the mods listed below.

* [Biome Makeover](https://www.curseforge.com/minecraft/mc-mods/biome-makeover)
* [Christmas Spirit](https://www.curseforge.com/minecraft/mc-mods/christmas-spirit-rewoven)
* [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
* [Fabric Simple Elevators](https://www.curseforge.com/minecraft/mc-mods/fabric-simple-elevators)
* [Joban Client Mod](https://modrinth.com/mod/jcm)
* [Minecraft Transit Railway](https://www.curseforge.com/minecraft/mc-mods/minecraft-transit-railway)
* [London Underground Addon](https://www.curseforge.com/minecraft/mc-mods/london-underground)
* [RGB Blocks](https://www.curseforge.com/minecraft/mc-mods/rgb-blocks)
